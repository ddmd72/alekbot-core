# 🧠 Alek-Core: Sovereign Exocortex

Alek-Core — это персональная система управления знаниями ("второй мозг"), построенная на принципах **Clean Architecture**, когнитивной психологии и современных LLM.

## 🚀 Быстрый старт

1.  **Установка:** Следуйте [Инструкции по установке](docs/guides/INSTALLATION.md).
2.  **Запуск:**
    ```bash
    make local
    ```
3.  **Деплой:**
    ```bash
    make deploy-dev
    ```

## 📚 Документация

Вся документация проекта организована в папке `docs/`. Начните с **[docs/README.md](docs/README.md)** для обзора всех разделов.
